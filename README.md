# PAL-CleanX: Autonomous Space Debris Cleanup

## 🚀 Overview
PAL-CleanX is an AI-powered robotic system designed to clean space debris in Low Earth Orbit (LEO). This project combines advanced orbital research with a physical hardware prototype of a 6-DOF capturing arm.

## 📁 Repository Structure
* **/docs**: Contains the Concept Document and Presentation PDF.
* **/hardware**: Contains the Arduino/Python source code and a photo of the robotic hand prototype.
* **/simulation**: Technical research files and orbital calculations.
* **/assets**: Demo video and project diagrams.

## 🛠 Technical Evidence
* **Hardware:** A physical 6-DOF robotic arm prototype has been developed to prove mechanical capture feasibility.
* **Software:** The system utilizes CNNs for debris detection and PPO for autonomous navigation.
* **Code:** You can find the control logic in the `/hardware` folder.
